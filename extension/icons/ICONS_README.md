# Extension Icons

For Chrome Web Store submission, you need 3 icon sizes:

- **icon16.png** - 16x16px (toolbar icon, small)
- **icon48.png** - 48x48px (extension management)
- **icon128.png** - 128x128px (Chrome Web Store)

## Quick Generation

You can use the included `generate-icons.html` file or create icons with any tool.

### Option 1: Use generate-icons.html (placeholder)
1. Open `extension/generate-icons.html` in browser
2. Download the generated icons
3. Place them in this `icons/` folder

### Option 2: Design custom icons
Use Figma, Canva, or any image editor to create:
- Simple, recognizable logo
- High contrast (works on light/dark backgrounds)
- Clear even at 16x16

### Option 3: Use online generators
- https://realfavicongenerator.net/
- https://www.favicon-generator.org/
